## Naukri Code360 Web Projects 2

This repository contains the Source Code of project 'News Paper Layout' I completed as part of the Naukri 360 Coding Ninjas Web Development practice questions. It includes HTML and CSS, and the kit was provided by the Coding Ninjas team. We modified it to achieve the design specified by them.


### How To Submit a Project

1. Download the starter kit.  
2. After downloading, extract the kit (zip File scaffold).  
3. Open the folder. Inside the scaffold folder, there is another scaffold folder—open it in VS Code.  
4. Design the page and complete the tasks given on the website.  
5. After completing, save the files.  
6. Open File Explorer and go to the extracted scaffold folder.  
7. Copy `index.html`, `index.css`, and the `image-elements` files that you just created.  
8. Open the scaffold ZIP file and go inside the scaffold folder.  
9. Delete the existing `index.html` and `index.css` files, then paste the copied files [index.html, index.php] from the extracted scaffold folder.  
10. Upload the File in Coding Ninas Website and Wait For Evalaution
